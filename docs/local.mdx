---
title: Run Acontext Locally
description: "Use `acontext-cli` to run Acontext locally."
---

## Install `acontext-cli`
```bash
curl -fsSL https://install.acontext.io | sh
```

## Start Acontext Server Locally
```bash
acontext docker up
```
Once it's done, you can the following endpoints enabled:
- Acontext Base URL: http://localhost:8029/api/v1
- Acontext Dashboard: http://localhost:3000/

## Try some Examples
### Interactive mode
```bash
acontenxt create
```
